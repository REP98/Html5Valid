Html5 Validación

Validador y manejador de formularios usuando las api de javascript y html5

HTML5Valid, es un plugins javascript que no permite validar cualquier formulario de forma sencialla y de manera automatica.

Instalación

Para instalarlo es muy sencillo solo necesita Descargar este plugin desde github y copiar la carpeta dist a su proyecto, luego copie y pege las lineas a continuación. 

							<!-- Agrege esto al head -->
							<link rel="stylesheet" type="text/css" href="../dist/css/html5valid.css">
							<!-- Agrege esto al final de Body -->
							<script src="../dist/js/html5valid.js"></script>
