Html5 Validación

Validador y manejador de formularios usuando las api de javascript y html5

HTML5Valid, es un plugins javascript que no permite validar cualquier formulario de forma sencialla y de manera automatica.

Instalación

Para instalarlo es muy sencillo solo necesita Descargar este plugin desde github y copiar la carpeta dist a su proyecto, luego copie y pege las lineas a continuación. 


							<!-- Agrege esto al head -->
							<link rel="stylesheet" type="text/css" href="../dist/css/html5valid.css">
							<!-- Agrege esto al final de Body -->
							<script src="../dist/js/html5valid.js"></script>

Opciones para el Formulario

						

							

								Opciones que solo se colocan dentro de los Dataset del elemento form
								
									

										Data-*
										Tipo
										Valores por Defecto
										Descripción
									

								
								
									

										novalidate
										boolean
										true
										Si es true anula la validación del navegador
									

									

										all-required
										booelan
										false
										Establece si todos los ampos son obligatorios
									

									

										cls
										string
										
										Class css para el formulario.
									

									

										clsValid
										string
										
										Class css para el formulario si es valido.
									

									

										clsInvalid
										string
										
										Class css para el formulario si es invalido.
									

									

										online
										boolean
										false										
										Permite establecer si los elementos seran validados en linea
									

									

										event
										string
										submit
										Permite establecer el evento que disparara el formulario
									

									

										add-cls-required
										boolean
										true
										Permite establecer si se debe usar una clase css para los elementos requeridos
									

									

										show-type-error
										string
										help
										Establece como debe ser mostrado los errores, solo tiene 2 valores posibles: help y notice
									

									

										help
										object
										
											

												{
												    icons:'fa fa-times',
												    cls:'help-invalid'
												}
												

												
										Si data-show-type-error es help esta opciones extilizan la apariensia de mensaje de ayuda
									

									

										notice
										object
										
											

												{
												    cls:'dange',
												    icons:'fa fa-times',
												    duration:5000,
												    onEscClickClose:true,
												}
												

												
										Si data-show-type-error es notice esta opciones extilizan la apariensia de las notificaciones
									

									

										xhr
										object or boolear
										{}
										Establece las opciones de Ajax
									

									

										rule
										object
										
											

												{
												    valido:'Formulario Valido',
												    invalido:'Formulario Invalido',
												    xhrError:'Error al enviar formulario',
												    text:"Por favor ingrese un texto valido",
												    email:'Ingrese una direccion de Correo valido',
												    url:'Ingrese una Url valida',
												    tel:'Ingrese un número de teléfono validos',
												    number:'Este campo solo acepta numeros',
												    color:'Campo de para colores',
												    date:'Ingrese una fecha valida',
												    datetime:'Ingrese una fecha y hora validas',
												    time:'Ingrese una hora valida',
												    seach:'Ingrese un texto valido',
												    password:'Ingrese una Contraseña valida',
												    equalTo:'No concuerdan las contraseñas',
												    different:'Son identicas',
												    ip:'IP invalida',
												    decimal:'No es un decimal Valido',
												    required:'Elemento Requerido'
												}
												

												
										Establece los mensajes que se mostraran si los campos según su tipo son invalidos
									

									

										on-success
										function
										
											

												(data, textStatus, jqXHR)=>{}
												

												
										Establece la funcion en caso de que la validacion a salido bien.
									

									

										on-error
										function
										
											

												(jqXHR, textStatus, error)=>{}
												

												
										Establece la funcion en caso de error.
